### 📥 Installation
1. Go to **[Releases](https://github.com/leopanorginal/MIAU/releases)** 
2. Download **`MIAU.Browser.Setup.5.0.0.exe`**
3. Run the installer
4. Choose where to install
5. Click install and enjoy! 🚀

### Features
- Privacy focused (adblock, tracker protection, HTTPS-only)
- 50+ themes (dark, light, cute, hacker, space)
- Hebrew language support
- Self-destruct mode
- Cute chibi girl animation 🥺
- Built-in PDF viewer
- Bookmarks & history
